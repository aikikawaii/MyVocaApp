    AI 다중 분석 그룹핑 🧩

    자연어 처리(NLP) + AI 의미 분석으로 단어들을 5가지 관점에서 심층 분석합니다.

    단어 분석 중...

    0 / 0

    0️⃣ 한국어 의미 클러스터 그룹핑 🧠

    입력한 한국어 뜻을 Claude AI가 직접 읽고 의미가 유사한 단어들을 묶어냅니다. 문자가 달라도 뜻이 비슷하면 같은 그룹이 됩니다.

    1️⃣ 어근 및 파생어 패밀리 (Lemmatization)

    자체 JS 형태소 분석기로 단어의 접미사를 분리해 같은 뿌리(Root)를 가진 단어를 묶습니다.

    2️⃣ 어원 및 접두사 매칭 (Prefix Rules)

    trans-, inter-, sub- 등 핵심 접두사 DB를 기반으로 패턴을 추출합니다.

    3️⃣ 유의어 / 반의어 연결망 (Semantic Graph)

    API가 제공하는 명확한 유의어/반의어 관계망에 내 단어장이 속해있는지 교차 검증합니다.

    4️⃣ 주제별 카테고리 (Topic Clustering)

    WordNet 상위 개념 API로 각 단어의 의미 계층을 추출해 17개 카테고리로 자동 분류합니다.
